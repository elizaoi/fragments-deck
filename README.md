# Question Cards Site

Пустой каркас одностраничного сайта для карточной игры.

## Команды

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Структура

- `src/App.tsx` - разметка страницы и placeholder-блоки.
- `src/App.css` - стили секций и адаптив.
- `src/index.css` - базовые переменные и глобальные стили.
