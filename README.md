# Фрагменты

Прототип архива-канваса для разработки колоды карточной игры, вдохновленной книгой Ролана Барта «Фрагменты речи влюбленного».

## Что уже есть

- 5 стартовых папок по 20 карт, всего 100 карт.
- Большой прокручиваемый канвас с перетаскиванием карт.
- Редактор выбранной карты: название, лицевая сторона, оборот, масть, папка.
- Переворот карты.
- Добавление и удаление карт.
- Создание, переименование и удаление папок.
- Сворачиваемая панель папок.
- Явный переключатель лицевой стороны и оборота карты.
- Сохранение состояния в `localStorage`.

## Команды

```bash
npm install
npm run dev
npm run build
npm run lint
```
